    Procédure de Déploiement - DIGIPLEX¶

    Schéma Logique du Réseau¶

    Architecture réseau détaillée de DIGIPLEX

    Plan d'Adressage¶

    VLANs et Sous-réseaux¶

    • VLAN 10 (Conception) : 192.168.10.0/24
    • VLAN 20 (Commercial) : 192.168.20.0/24
    • VLAN 30 (Ressources Humaines) : 192.168.30.0/24
    • VLAN 40 (Hotline) : 192.168.40.0/24
    • VLAN 50 (Wifi Enterprise) : 192.168.50.0/24
    • VLAN 60 (Wifi invités) : 192.168.60.0/24
    • VLAN 70 (Serveurs) : 192.168.70.0/24
    • VLAN 80 (Management) : 192.168.80.0/24

    Interconnexion FAI¶

    • Lien DSLAM : 172.16.4.0/30
    • Router DIGIPLEX : 172.16.4.1
    • FAI : 172.16.4.2

    Configurations des Équipements¶

    Switch L3 Core (DIGI0_SWL3_01)¶

    hostname DIGI0_SWL3_01
!
ip routing
!
! VTP Configuration
vtp domain digiplex.switch.com
vtp mode server
vtp version 2
!
! VLANs
vlan 10
 name CONCEPTION
vlan 20
 name COMMERCIAL
vlan 30
 name RH
vlan 40
 name HOTLINE
vlan 50
 name WIFI_ENTERPRISE
vlan 60
 name WIFI_INVITE
vlan 70
 name SERVEURS
vlan 80
 name MANAGEMENT
!
! Etherchannel Trunks
interface Port-channel1
 switchport trunk encapsulation dot1q
 switchport mode trunk
!
interface Port-channel2
 switchport trunk encapsulation dot1q
 switchport mode trunk
!

    Router (DIGI0_RT_01)¶

    hostname DIGI0_RT_01
!
interface FastEthernet0/0
 ip address 192.168.70.200 255.255.255.0
!
interface GigabitEthernet0/0/0
 ip address 172.16.4.1 255.255.255.252
!
ip route 0.0.0.0 0.0.0.0 172.16.4.2

    Switch Accès Type (DIGI0_SWL2_XX)¶

    hostname DIGI0_SWL2_XX
!
vtp domain digiplex.switch.com
vtp mode client
!
! Trunk vers Core
interface FastEthernet0/1
 switchport mode trunk
 channel-group 1 mode active
!
interface FastEthernet0/2
 switchport mode trunk
 channel-group 1 mode active
!
! Ports VLAN 10 (Conception)
interface range FastEthernet0/3-8
 switchport mode access
 switchport access vlan 10
!
! Ports VLAN 20 (Commercial)
interface range FastEthernet0/9-14
 switchport mode access
 switchport access vlan 20
