    Projet FunkyTown - Documentation

    Présentation du Projet

    Ce document présente la documentation complète du projet FunkyTown, incluant la planification, l'infrastructure, les procédures de déploiement et la maintenance.

    Guide de Lecture des Diagrammes

    Diagramme PERT

    Le diagramme PERT utilise une représentation circulaire pour chaque tâche : 

    ┌─────────────[ID Tâche]─────────┐
│                                │
│   [Date début]    [Date fin]   │
│                                │
└──────────[Durée]──────────────┘
    - ID Tâche : Identifiant unique de la tâche - Date début : Date au plus tôt de début - Date fin : Date au plus tard de fin - Durée : Durée en jours de la tâche - Couleurs : - Rouge : Tâche critique (pas de marge) - Vert : Tâche avec marge

    Diagramme GANTT

    Le diagramme GANTT présente un planning temporel : 

    Tâche A  ████████
Tâche B      ████████
Tâche C          ████
    - Axe horizontal : Temps (jours) - Axe vertical : Tâches - Barres : Durée des tâches - Flèches : Dépendances - Couleurs : - Rouge : Tâche critique - Bleu : Tâche normale - Gris : Marge disponible

    Diagrammes Réseau

    Les diagrammes réseau utilisent les conventions suivantes : 

    [R1]──────[SW1]
  │         │
  │         │
[R2]──────[SW2]
    - Rectangles : Équipements réseau - Lignes : Connexions physiques - Pointillés : Tunnels/VLANs - Couleurs : - Bleu : IPv6 - Vert : VLANs - Rouge : Liens critiques

    Vue d'Ensemble

    FunkyTown est un projet de déploiement réseau multi-sites comprenant : - 1 Datacenter central (IPv6) - 4 sites distants : - ESN eXia - Bibliothèque - Engie - Digiplex
