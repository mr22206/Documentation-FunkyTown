    Datacenter FunkyTown¶

    Vue d'ensemble¶

    Le Datacenter est le point central de l'infrastructure réseau de FunkyTown. Il héberge l'ensemble des services critiques et assure l'interconnexion sécurisée de tous les sites.

    Emplacement stratégique du Datacenter dans FunkyTown

    Infrastructure Technique¶

    Connectivité¶

    • Lien DSLAM FAI : 172.16.5.0/30
    • Tunnel IPv6 eXia : 2001:DB8:3000::/127
    • Réseau Cloud : 2001:DB8:1000::/64

    Services Centralisés¶

    DNS¶

    • Service de résolution centralisé pour tous les sites
    • Configuration spéciale : www.google.com -> 108.177.127.139
    • Gestion des zones internes pour les services locaux

    DHCP¶

    • Attribution dynamique des adresses IP
    • Configuration des options DNS par site
    • Gestion centralisée des baux

    Cloud¶

    • Services accessibles en IPv6 natif
    • Tunnel 6to4 dédié avec eXia
    • Applications et stockage distribués

    Interconnexions Sites¶

    Liens Dédiés¶

    • eXia : 172.16.1.0/30
    • Bibliothèque : 172.16.2.0/30
    • Engie : 172.16.3.0/30
    • Digiplex : 172.16.4.0/30

    Réseau Maillé¶

    • Adressage : 172.17.0.0/27
    • Capacité : 30 hôtes (25 requis)
    • Usage : Redondance et optimisation du routage inter-sites

    Sécurité et Haute Disponibilité¶

    Protection¶

    • Sécurisation physique multi-niveaux
    • Contrôle d'accès strict aux équipements
    • Monitoring permanent des services
    • Journalisation des événements

    Redondance¶

    • Liens redondants entre tous les sites
    • Équipements en configuration haute disponibilité
    • Systèmes de sauvegarde automatisés
    • Plan de reprise d'activité
