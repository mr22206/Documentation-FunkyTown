    Guide de Lecture¶

    Structure du Diagramme¶

    Le diagramme GANTT est un planning temporel qui se lit de gauche à droite : 

    Semaine 1     Semaine 2     Semaine 3
│             │             │
Tâche A  ████████
Tâche B      ████████
Tâche C          ████

    Éléments du Diagramme¶

    1. Axe Horizontal
    2. Représente le temps
    3. Échelle : Jours/Semaines

    4. Format date : JJ/MM

    5. Axe Vertical

    6. Liste des tâches
    7. Groupées par section

    8. Code couleur par type

    9. Barres de Tâches

    10. Longueur = Durée
    11. Position = Date début

    12. Couleur selon criticité :

      • Rouge : Tâche critique
      • Bleu : Tâche normale
      • Gris : Marge disponible

    13. Dépendances

    14. Flèches entre tâches
    15. Indiquent l'ordre d'exécution
    16. Types : Fin-Début, Début-Début, etc.
